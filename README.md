# Projet NIRD - Numérique Inclusif, Responsable & Durable

## URL du projet

[https://nuitinfo2025.jrcan.dev](https://nuitinfo2025.jrcan.dev)

---

## Description

Ce projet est un site web React qui promeut un numérique éducatif libre et responsable.  
Il intègre un header fixe, un logo, un slogan dynamique, et plusieurs sections accessibles via la navigation.

L’objectif est de sensibiliser à l’inclusion, la responsabilité et la durabilité dans le numérique, particulièrement dans les établissements scolaires.

---

## Fonctionnalités réalisées

- Header fixe   
- Navigation par ancres vers les sections (Accueil, La Démarche, Agir, Ressources, Communauté)  
- Logo avec effet au survol
- Section slogan avec design dynamique
- CSS organisé par composants
- Jeu interactif et version rétro

---

## Installation

1. Cloner le dépôt ou récupérer les fichiers sources.

2. Assurez-vous d’avoir [Node.js](https://nodejs.org/) et npm installés.

3. Installer les dépendances avec :

bash
npm install
